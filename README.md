**Predicción de Precios de Airbnb con Flask**

Este proyecto implementa un modelo de Machine Learning desarrollado para predecir los precios de propiedades en Airbnb utilizando Python y Flask. Se ha integrado un modelo de regresión previamente entrenado para estimar los precios basándose en múltiples características de las propiedades y su historial. El modelo y su entrenamiento se encuentran en el siguiente elnace(https://github.com/AndresPerelmuter/Tesis-Master-en-Big-Data)

**Características principales:**
- Interfaz web intuitiva: permite a los usuarios ingresar datos y obtener predicciones en tiempo real.
- Visualización interactiva: incluye gráficos dinámicos para explorar la densidad de precios estimados y reales.
